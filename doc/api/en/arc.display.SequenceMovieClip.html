SequenceMovieClip

A class that plays an animation by using multiple Image objects.
  • Extends
  • arc.display.DisplayObjectContainer

new SequenceMovieClip(imgArr, frameTime, shouldLoop, shouldHide)

Parameters:
Name Type Description
imgArr Array An array of Image objects
frameTime Number A millisecond of each frame
shouldLoop Boolean If true the animation will be looped
shouldHide shouldHide If true it will be hidden while the animation stops
